Ontologies and Software

The following OWL vocabularies and datasets have been developed in the scope of this project.

The whole production workflow for the ontology of Nouns and Verbs of the Gallo-Italic variety spoken in Nicosia and Sperlinga, aside with all the software tools and intermediate products, is available in the workflow page.

In addition, the Gallo-Siciliani github repository provide all the source codes of these tools.
